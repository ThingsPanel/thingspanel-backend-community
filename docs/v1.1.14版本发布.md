# ThingsPanel Backend Community v1.1.14 发布说明

**发布日期**：2026-04-02
**版本范围**：v1.1.13.6 (2026-03-21) → v1.1.14

---

## 新功能

### 设备模板增强
- **设备模板发布/订阅流程优化**：新增 `DeviceConfig` 和 `TemplateDefinition` 负载，提升设备模板的发布和订阅流程完整性
- **设备模板类型字段**：设备模板新增 `type` 字段，与 `brand`（品牌）和 `modelNumber`（型号）共同构成更完整的模板元数据描述

### 数据采集与聚合
- **遥测数据聚合查询优化**：对聚合查询中的 NULL 值和超大数值进行过滤，解决长时间窗口聚合时因异常数据导致的溢出问题

---

## 功能优化

### ThingsVis 可视化集成
- 新增 ThingsVis 集成 SQL 初始化脚本（版本 0.0.17），包含 ThingsVis 相关的表结构和基础数据
- 清理 ThingsVis SQL 初始化脚本，移除注释示例和冗余 INSERT 语句

### PostHog 遥测
- 优化 PostHog 安装遥测的生命周期管理，提升实例使用数据、版本及系统信息的采集准确性

---

## Bug 修复

- 修复设备激活时未自动绑定默认根组的问题

---

## 其他改动

### 配置与开发体验
- 新增开发环境配置文件（`.env.dev`）和演示环境配置文件
- 新增 `.instance_id` 本地配置文件管理
- 忽略并取消追踪 `.DS_Store` 文件和本地配置覆盖文件
- 忽略本地配置覆盖文件（覆盖 `.env` 的本地配置）
